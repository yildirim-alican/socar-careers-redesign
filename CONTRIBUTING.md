# Katkı Rehberi

Bu depo SOCAR Türkiye Kariyer sitesi için hazırlanmış bir **tasarım önerisi / prototiptir**. Resmî SOCAR ürünü değildir.

## Katkı kabulü

- Kurum içi değerlendirme ve geri bildirim için issue açabilirsiniz.
- Dış katkılar (pull request) yalnızca depo sahibinin onayı ile kabul edilir.
- SOCAR markası, logosu ve görselleri üzerinde değişiklik yapmadan önce kurumsal onay gereklidir.

## Geliştirme

1. Depoyu klonlayın.
2. Değişiklikleri UTF-8 uyumlu bir editörle yapın (PowerShell toplu düzenleme Türkçe karakterleri bozabilir).
3. CSS veya JS değişikliğinde tüm HTML dosyalarında `?v=` sürüm parametresini artırın.
4. Navbar/footer değişikliklerini **7 sayfanın tamamında** senkron tutun.

## Issue şablonları

Hata bildirimi veya iyileştirme önerisi için GitHub Issues kullanın; uygun şablonu seçin.
